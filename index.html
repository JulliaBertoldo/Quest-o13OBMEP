Questão 13 - OBMEP Nível 3

Enunciado

Os quadradinhos da figura devem ser preenchidos com os números de 1 a 7, de modo que:

  • os três números da linha horizontal fiquem em ordem crescente da esquerda para a direita;
  • os três números em cada coluna vertical fiquem em ordem crescente de cima para baixo.

De quantas maneiras é possível fazer esse preenchimento?

Resposta correta: 15 maneiras.

Resolução Matemática (OBMEP)

Nomeamos as posições:

A B C
D   E
F   G

As condições são:

A < B < C
A < D < F
C < E < G

A precisa ser menor que todos os outros números. Logo: 

A = 1

Restam os números: 

2,3,4,5,6,7

Sem restrições haveria: 

6! = 720

As condições: 

B < C
D < F
E < G

eliminam metade das possibilidades em cada par: 

720 ÷ 2 ÷ 2 ÷ 2 = 90

Ainda precisamos garantir: 

B < C < E < G

Das 90 distribuições, apenas 1/6 respeita essa ordem completa: 

90 ÷ 6 = 15

Resposta:

15 maneiras

Resolução Computacional

O algoritmo:

  1. Gera todas as permutações dos números 1 a 7.
  2. Existem 7! = 5040 permutações.
  3. Verifica as condições:
A < B < C
A < D < F
C < E < G

Se todas forem verdadeiras, a configuração é válida.

Ao final o contador retorna: 

15
